1. 통합된 데이터  
2 .저장된 데이터
3. 운영 데이터

DBMS(DATABASE Management system)
data 무결성을 높여줌(신뢰성을 높여줌)
data 중복, 동시 사용 불가능 문제를 해결해줌

DDL (data definition language)

DCL (Control) - 데이터 제어
CRUD (create, retrieval, upgrade, delete) = DML
